Tic Tac Toe – WinForms-basierte Spielanwendung

Projektziel

Das Ziel dieses Projekts war die Entwicklung einer grafischen Umsetzung des klassischen Spiels Tic Tac Toe mit Hilfe von C# und WinForms. Es sollte eine einfache Benutzeroberfläche entstehen, die ein reibungsloses Zwei-Spieler-Spiel auf einem Gerät ermöglicht.

Eingesetzte Technologien

  • Programmiersprache: C#
  • UI-Technologie: Windows Forms (WinForms)
  • Entwicklungsumgebung: Visual Studio

Umsetzung und Besonderheiten

Die Spiellogik wurde vollständig manuell implementiert. Das Spiel erkennt automatisch gültige Züge, Siegbedingungen, Unentschieden und blockiert weitere Züge nach Spielende. Beim Gewinn erscheint ein kleines Pop-up-Fenster, das den Gewinner anzeigt. Die Benutzeroberfläche wurde mithilfe von WinForms erstellt und reagiert auf Spielaktionen.

Besonderes Augenmerk lag auf der klaren Trennung von Spiellogik und UI, um die Wartbarkeit und Erweiterbarkeit zu gewährleisten.

Mein Beitrag

Ich habe das Projekt vollständig eigenständig umgesetzt – von der Idee über die Planung bis zur Programmierung der Spielmechanik. Ziel war es, meine Kenntnisse in C# zu vertiefen. WinForms war meine erste grafische Oberfläche, die ich durch Mitschüler kennengelernt hatte.

Herausforderungen

Die größte Herausforderung war die saubere Spiellogik, die alle Spielverläufe korrekt erkennt und zuverlässig auf Benutzeraktionen reagiert. Es war mein erstes größeres Projekt mit einer grafischen Benutzeroberfläche.

Codebeispiel: Gewinnerermittlung & Spielauswertung


// Prüft alle Gewinnkombinationen
public void checkForWinner()
{
    for (int i = 0; i < 8; i++)
    {
        string combo = i switch
        {
            0 => spielbrett[0] + spielbrett[4] + spielbrett[8],
            1 => spielbrett[2] + spielbrett[4] + spielbrett[6],
            2 => spielbrett[0] + spielbrett[1] + spielbrett[2],
            3 => spielbrett[3] + spielbrett[4] + spielbrett[5],
            4 => spielbrett[6] + spielbrett[7] + spielbrett[8],
            5 => spielbrett[0] + spielbrett[3] + spielbrett[6],
            6 => spielbrett[1] + spielbrett[4] + spielbrett[7],
            7 => spielbrett[2] + spielbrett[5] + spielbrett[8],
            _ => ""
        };

        if (combo == "xxx")
            MessageBox.Show("X hat das Spiel gewonnen!");
        else if (combo == "000")
            MessageBox.Show("0 hat das Spiel gewonnen!");
    }

    checkDraw();
}

// Prüft auf Unentschieden
public void checkDraw()
{
    if (spielbrett.All(field => field != null))
        MessageBox.Show("Unentschieden – kein Gewinner.");
}
