Bibliotheksverwaltung – Konsolenanwendung mit OOP

Projektziel

Ziel dieses Projekts war die Entwicklung einer einfachen Konsolenanwendung zur Verwaltung einer kleinen Bibliothek. Benutzer sollten Bücher ausleihen, zurückgeben und das Programm beenden können. Der Fokus lag auf der praktischen Anwendung objektorientierter Programmierung in C#.

Eingesetzte Technologien

  • Programmiersprache: C# mit objektorientierten Programmierung
  • Technologie: .NET-Konsole
  • Entwicklungsumgebung: Visual Studio

Umsetzung und Besonderheiten

Die Anwendung basiert auf mehreren Klassen wie Buch, Benutzer und Bibliothek, die zusammen eine einfache, aber funktionale Bibliotheksverwaltung ermöglichen. Bücher können ausgeliehen und zurückgegeben werden. Die Interaktion erfolgt vollständig über die Konsole.

Der Fokus lag auf der korrekten Anwendung von OOP-Prinzipien wie Kapselung, Vererbung und Polymorphie. Dadurch wurde das Projekt übersichtlich und leicht erweiterbar gestaltet.

Mein Beitrag

Ich habe die komplette Anwendung selbst geplant und programmiert. Ziel war es, meine Kenntnisse im Bereich der objektorientierten Programmierung zu vertiefen und eine sinnvolle Konsolenanwendung zu erstellen, die reale Prozesse simuliert.

Herausforderungen

Die größte Herausforderung war die saubere Klassenstruktur sowie das fehlerfreie Zusammenspiel der Methoden beim Ausleihen und Zurückgeben von Büchern. Außerdem war es eine gute Übung, Benutzerinteraktionen sinnvoll und benutzerfreundlich über die Konsole zu gestalten.

Codebeispiel: Buch ausleihen & zurückgeben


// Buch ausleihen
  public void BuchAusleihen(string isbn, int mitgliedsnummer)
  {
      var buch = SucheBuchNachISBN(isbn);
      if (buch != null && !buch.IstAusgeliehen)
      {
          buch.IstAusgeliehen = true;
          Console.WriteLine($"Das Buch '{buch.Titel}' wurde erfolgreich ausgeliehen.");
      }
      else
      {
          Console.WriteLine("Das Buch ist nicht verfügbar.");
      }
  }

  public void BuchZurueckgeben(string isbn)
  {
      var buch = SucheBuchNachISBN(isbn);
      if (buch != null && buch.IstAusgeliehen)
      {
          buch.IstAusgeliehen = false;
          Console.WriteLine($"Das Buch '{buch.Titel}' wurde erfolgreich zurückgegeben.");
      }
      else
      {
          Console.WriteLine("Das Buch wurde nicht ausgeliehen.");
      }
    }
